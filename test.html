Restore Wallet
To restore wallet you must have not seed file in wallet_data folder and not epic-wallet.toml file in folder where you will restore wallet. For Default option it is .epic/main folder and for Local it is application folder. Prepare your restore phrase and new password. Use it in command console. After Restore success please login and start checking which take a while so time for a cup of coffee.
